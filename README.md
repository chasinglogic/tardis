## Building

The following dependencies are required to build Tardis:

    - libgtk-3-dev
    - libjson-glib-dev
    - libgranite-dev
    - rsync
    - meson
    - valac >= 0.40.3

Use the following commands to compile Tardis:

```
meson build
ninja -C build
```

To install your freshly built version:

```
sudo ninja -C build install
```
